Description:-
This is an engaging, tile-based memory game designed to test and improve your cognitive skills. Players must flip tiles to reveal symbols or numbers and find matching pairs. The goal is to clear the board in the fewest moves possible, making it a perfect quick-play experience for casual gamers.

Tech Stack:-

1. Language: Python
2. GUI Framework: Tkinter
3. Logic: Standard Python libraries for randomizing and handling game states.

Inspiration & Motivation:-
The inspiration came from the classic "Concentration" or "Memory" card game. We wanted to build a lightweight, desktop-based GUI application that demonstrates how event-driven programming can create interactive and fun user experiences. This project was born out of a desire to practice UI design in Python while creating something playable and rewarding.

How to Run:-
1. Ensure you have Python installed.
2. Download the Python_Gui_Game.exe file provided.
3. Double-click the file to launch the game directly without needing a Python environment.
